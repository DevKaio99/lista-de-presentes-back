# 🎁 Lista de Presentes – Backend

API REST desenvolvida para gerenciar uma **lista de presentes de chá de casa nova**.

Esta aplicação é responsável por fornecer os dados utilizados pelo frontend, permitindo consultar, adicionar e atualizar os presentes disponíveis na lista.

O projeto foi desenvolvido como **projeto pessoal full stack**, com o objetivo de praticar desenvolvimento backend, criação de APIs REST, integração com frontend e conteinerização da aplicação.

---

# 📌 Visão Geral do Projeto

O backend fornece uma API responsável por gerenciar os presentes disponíveis para um evento de chá de casa nova.

A aplicação permite:

- Listar os presentes disponíveis
- Cadastrar os usuários e os presentes escolhidos
- Atualizar o status de um presente (disponível ou atualização da quantidade)

O frontend consome essa API para exibir e atualizar os dados da lista de presentes.

⚠️ Observação: A aplicação já foi hospedada anteriormente, porém atualmente o backend não está em execução online.

---

# 🚀 Funcionalidades

- Listagem de presentes
- Cadastro dos presenteadores e os respectivos presentes escolhidos
- Atualização do status de um presente
- Integração com frontend via API REST
- Persistência de dados em banco PostgreSQL

---

# 🛠️ Tecnologias Utilizadas

- Java
- Spring Boot
- Arquitetura MVC (Model-View-Controller)
- Docker para conteinerização
- PostgreSQL como banco de dados
- Neon (plataforma de hospedagem de PostgreSQL)
- API REST
